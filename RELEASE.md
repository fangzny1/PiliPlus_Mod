# PiliPlus Mod v2.1.0

基于上游 PiliPlus v2.1.0 合并，保留全部 MOD 功能。

## 🆕 Mod 特色功能

### 主动发现推荐引擎
- **关注UP采样** — 随机抽取 1~5 个关注 UP，拉取投稿视频
- **相关视频展开** — 从采样视频展开多路相关推荐
- **混合注入** — 发现视频按间隔混入手机端推荐流
- **自动去重** — 跨种子、跨 UP 全局 BVID 去重
- **已看过滤** — 自动比对观看记录，排除已看内容

### 发现推荐可调参数
在 **设置 → 推荐设置 → 主动发现推荐** 中：
| 参数 | 范围 | 说明 |
|------|------|------|
| 采样UP数 | 1~5 | 每次刷新生效的UP数量 |
| 每批总量 | 10~40 | 注入推荐流的发现视频数 |
| 混入间隔 | 3~10 | 越小发现视频越多 |
| 相关视频展开 | 开关 | 是否展开相关推荐 |

### 兴趣标签系统
- **手动添加** — 视频详情菜单 → 添加兴趣标签 → 弹窗勾选
- **自动记录** — 设置中可开启，打开视频自动收录标签
- **权重管理** — 相同标签重复收录自动叠加权重
- **管理页面** — 查看、编辑、删除、清空标签

### "此类内容过多" 一键过滤
- 视频三点菜单新增选项
- 一键拉黑 UP + 提交不感兴趣 + 移除卡片
- 自动弹出标题关键词拆词器，勾选或手动编辑

### 其他改进
- 修复"看到这里"标记不显示的问题
- 合并上游 v2.1.0 全部改动
- 标题关键词过滤 UI 优化（可滚动、可编辑文本框）
- GitHub Actions 自动构建（arm64-v8a / armeabi-v7a / x86_64）

## 📝 使用方法

1. 安装后打开 → **设置 → 推荐设置**
2. 开启 **主动发现推荐** — 首页推荐流自动混入发现视频
3. 调整 **采样UP数** 和 **混入间隔** 控制发现比例
4. 可选开启 **兴趣标签搜索** — 手动或自动收录标签
5. 遇到不喜欢的视频 → 三点菜单 → **此类内容过多**
